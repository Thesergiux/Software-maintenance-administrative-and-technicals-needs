# Sistema de Gestión de Tareas con PostgreSQL

**Requisitos**
- Python 3.12+
- PostgreSQL 17
- Librerías: `psycopg2-binary`, `uuid`

**Dependencias**:
pip install psycopg2-binary

**Migración de Datos**
Para migrar datos desde tasks.json:
    python migrate.py
